PLMS — Parking Lot Management System
Expo SDK 54 · React Native 0.81 · FastAPI · MongoDB

Cross-platform mobile app (Android APK via Expo EAS) sharing a single FastAPI + MongoDB backend with a web admin dashboard. Built for gate security officers to manage vehicle entries, events, and staff schedules in real time.

  • File-based routing with expo-router; Bottom Tab + Stack navigation via React Navigation 7.x
  • FastAPI backend with Pydantic v2, Motor async driver, CORS middleware, and Kubernetes ingress routing /api/* to port 8001

  • Single MongoDB source of truth (parking_management DB) shared across mobile and web clients
  • 15 passing integration tests with pytest; Playwright used for frontend E2E during dev
  • Stateless backend with bearer tokens; AsyncStorage for mobile auth persistence
